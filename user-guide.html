Method 1: Admin Panel (Recommended)

Best for: Beginners, quick user management, bulk operations
Open the Admin Panel: Navigate to admin.html in your browser
Add Single User: Fill in the email, password, and optional name fields
Bulk Import: Use the JSON format to add multiple users at once
View Users: See all current users in the user list below
Delete Users: Click the delete button next to any user

Example JSON for Bulk Import:

[ { "email": "john@example.com", "password": "securePass123", "name": "John Doe" }, { "email": "jane@example.com", "password": "myPassword456", "name": "Jane Smith" } ]

Method 2: Browser Console Commands

Best for: Developers, quick testing, programmatic access
Open Console: Press F12 or right-click ‚Üí Inspect ‚Üí Console tab
Copy the code below and paste it into the console
Press Enter to execute

Add Single User:

// Add a single user const users = JSON.parse(localStorage.getItem('users')) || []; users.push({ email: 'newuser@example.com', password: 'securepassword123', name: 'New User', created: new Date().toISOString() }); localStorage.setItem('users', JSON.stringify(users)); console.log('User added successfully!');

Add Multiple Users:

// Add multiple users at once const newUsers = [ {email: 'user1@example.com', password: 'pass1', name: 'User 1'}, {email: 'user2@example.com', password: 'pass2', name: 'User 2'}, {email: 'user3@example.com', password: 'pass3', name: 'User 3'} ]; const existingUsers = JSON.parse(localStorage.getItem('users')) || []; const updatedUsers = [...existingUsers, ...newUsers]; localStorage.setItem('users', JSON.stringify(updatedUsers)); console.log(`${newUsers.length} users added successfully!`);

View All Users:

// View all stored users const users = JSON.parse(localStorage.getItem('users')) || []; console.table(users);

Method 3: Manual localStorage Editing

Advanced Method: Requires basic JavaScript knowledge
Open Browser DevTools: Press F12 or right-click ‚Üí Inspect
Go to Application/Storage tab: Find localStorage section
Locate 'users' key: Click on it to edit the value
Edit the JSON array: Add your user objects
Save changes: The changes are applied immediately

Example localStorage Format:

[ { "email": "existing@example.com", "password": "oldpass", "name": "Existing User", "created": "2024-01-01T00:00:00.000Z" }, { "email": "newuser@example.com", "password": "newpass123", "name": "New User", "created": "2024-01-15T10:30:00.000Z" } ]

Method 4: Bulk Import from File

Best for: Large user bases, data migration, automated processes
Create a JSON file: Save user data as users.json
Use Admin Panel: Go to admin.html ‚Üí Bulk Import tab
Paste JSON data: Copy from your file and paste
Import users: Click the Import Users button

Sample users.json file:

{ "users": [ { "email": "admin@website.com", "password": "admin123", "name": "Administrator", "role": "admin" }, { "email": "user1@website.com", "password": "user123", "name": "Regular User 1" }, { "email": "user2@website.com", "password": "user456", "name": "Regular User 2" } ] }

üîß Troubleshooting

Common Issues:

  • Login not working: Check if user exists in localStorage
  • Users not saving: Ensure localStorage is not disabled
  • JSON errors: Validate your JSON syntax
  • Duplicate emails: Each user must have a unique email

Debug Commands:

// Check if users exist console.log('Total users:', JSON.parse(localStorage.getItem('users') || '[]').length); // View current user console.log('Logged in user:', localStorage.getItem('loggedInUser')); // Clear all data (CAUTION!) localStorage.clear();

üîí Security Considerations

Important: This is a client-side only system for demonstration purposes.

Production Recommendations:

  • Use server-side authentication (Node.js, PHP, etc.)
  • Hash passwords with bcrypt or similar
  • Implement proper session management
  • Use HTTPS for secure data transmission
  • Regular security audits and updates
